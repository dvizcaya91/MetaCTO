## UI & UX Style guidelines

Trendy 2025 Web App (Minimal • High-Contrast • Electric Blue • 3D • Dark Mode • Glassmorphism)

Create a modern web app UI that feels cutting-edge and trendy for 2025. The interface should be minimalist, bold, and futuristic—high clarity, high contrast, and purposeful use of depth.

1) MINIMALISM (clean + intentional)
- Use a strict grid and strong alignment. Everything has a reason to exist.
- Generous negative space; avoid visual clutter and excessive decoration.
- Simple, confident typography (large headings, clean body text, strong hierarchy).
- UI elements are “quiet” by default; emphasis comes from contrast, spacing, and micro-interactions.

2) DARK MODE (default)
- Dark mode is the primary experience (not an afterthought).
- Backgrounds: deep charcoal / near-black with subtle tonal variation (no flat pure black everywhere).
- Text: high legibility (near-white for primary text, soft gray for secondary).
- Use soft ambient lighting and subtle glows to create depth without noise.

3) HIGH CONTRAST + ELECTRIC BLUE (signature accent)
- Core palette: dark neutrals + electric blue as the hero accent.
- Electric blue is used for: CTAs, focus states, active navigation, key highlights, data emphasis.
- Keep other bright colors minimal; if used, only as tiny supporting accents.
- Add restrained “energy” via blue glows, edge highlights, or thin gradient strokes (never overdone).

4) GLASSMORPHISM (layered, frosted, premium)
- Use frosted glass panels for nav bars, modals, side panels, and floating cards.
- Properties: translucency + background blur + subtle border (1px) + soft shadow.
- Keep legibility sacred: ensure text/icon contrast stays high over glass surfaces.
- Glass should feel like layered UI depth, not decoration—use it where hierarchy benefits.

5) 3D OBJECTS (tasteful, not gimmicky)
- Integrate 3D elements sparingly in high-impact zones: hero section, empty states, feature highlights.
- 3D objects should feel “product-like”: smooth materials, soft studio lighting, subtle reflections.
- Use depth cues: gentle shadows, parallax, slight rotation on hover, slow drift animations.
- If interactive 3D is heavy, use static renders or subtle pseudo-3D to keep performance fast.

LAYOUT + COMPONENT GUIDELINES
- Cards: minimal content density, rounded corners, glass option, subtle elevation.
- Buttons: bold primary in electric blue; secondary as outline/glass; clear hover/pressed states.
- Inputs: clean, minimal borders; strong focus ring in electric blue; comfortable padding.
- Navigation: simple, sticky top bar with glassmorphism; clear active states.
- Data UI: high contrast charts/indicators with electric blue highlights; avoid busy grids.

MOTION + INTERACTIONS (smooth, premium)
- Micro-interactions: hover glow, subtle scale (tiny), smooth easing, soft transitions (150–250ms).
- Scroll behavior: gentle parallax for 3D elements; progressive reveal (subtle fades/slide-up).
- Feedback: clear states (loading, success, error) with minimal but confident cues.

MATERIAL + FINISH (overall vibe)
- “Futuristic but usable” — premium, clean, calm, confident.
- Use soft gradients only when they add depth (e.g., blue glow gradients, glass highlights).
- Avoid noisy textures; rely on light, blur, shadow, and spacing as the main visual language.

COLOR TOKENS (example targets, adjust as needed)
- Background: #0A0A0A to #121212 (with slight tonal panels)
- Surface: rgba(255,255,255,0.06) for glass surfaces
- Border: rgba(255,255,255,0.10)
- Primary Accent (electric blue): #2D6BFF / #1E90FF / #00A3FF (choose one primary)
- Glow: electric blue at low opacity (10–25%)
- Text Primary: #F2F4F8
- Text Secondary: #A8B0BC

FINAL CHECKS
- Minimal, clean, and spacious—yet bold via contrast and electric blue accents.
- Dark mode + glass layers + tasteful 3D accents.
- No clutter, no gimmicks, high performance, premium feel.
